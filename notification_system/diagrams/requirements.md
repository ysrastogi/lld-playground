# Notification System

# Functional Requirements
- MultiChannel Support
- Notification Delivery
- Rate Limiting
- Retry Mechanism
- Application Events

# Non Functional Requirements
- Scalability
- Availability
- Fault Tolerance
- Low Latency
- Durability

# System Requirements
- Support 1M+ Users
- Support 1000+ Notifications per second
- Support 100+ Notifications per user per day
