# System Requirement
## Overview
The **URL Shortener** is a web service that allows users to shorten long URLs to shorter URLs. 

## Functional Requirements

- Users can shorten a URL.
- Users can redirect a shortened URL to the original URL.
- Users can delete a shortened URL.
- Users can update a shortened URL.
- Users can stale a shortened URL.

## Non-Functional Requirements

- The service should be highly available.
- The service should be scalable.
- The service should be fault-tolerant.
- The service should be secure.
- The service should be performant.

## Scale

- 100M URLs
